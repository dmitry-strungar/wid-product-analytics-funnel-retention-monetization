# WID Product Analytics: Воронка, Retention и Монетизация

## Обзор проекта

Этот проект представляет собой комплексный анализ продуктовой аналитики цифрового сервиса WID.  
Анализ сосредоточен на эффективности пользовательской воронки, удержании пользователей, эффективности каналов привлечения и A/B-тестировании модели монетизации.

Данный проект представляет собой реальную задачу анализа продукта, компании было необходимо выявить возможности для роста доходов и оптимизировать конверсию пользователей.

---

## Бизнес-контекст

Компания стремится:

- Выявить точки потерь в пользовательской воронке  
- Повысить уровень удержания пользователей  
- Оценить эффективность каналов привлечения  
- Протестировать новую модель монетизации  

---

## Цели проекта

- Построить пользовательскую воронку (Visit → Registration → Activity → Subscription)  
- Рассчитать DAU / MAU / WAU  
- Выполнить когортный анализ (Retention 7, Retention 30)  
- Сравнить маркетинговые каналы (Google Ads, TikTok, Instagram)  
- Провести статистический A/B-тест гипотезы монетизации  

---

## Аналитический подход

- Анализ воронки и расчет конверсии  
- Когортный анализ удержания  
- Сравнение маркетинговых каналов  
- Расчет сегментационных метрик  
- Формирование гипотез и проверка статистической значимости  

---

## Ключевые инсайты

- Основная точка потерь пользователей — этап монетизации  
- Более высокая частота использования сервиса коррелирует с вероятностью подписки  
- TikTok демонстрирует более высокие показатели монетизации  
- Новая модель монетизации показывает положительный потенциал, однако требует увеличения выборки для подтверждения статистической значимости
